HOW TO RUN:
1) Install Python if it is not installed
2) Install Django if it is not installed: "pip install django"
3) Open a command prompt and navigate to the project directory
4) Run "python manage.py collectstatic" and then "yes"
4) Start Django project using the following command: "python manage.py runserver"
5) Once the server is running, open a web browser and go to the address where Django project is running. By default, it will be http://127.0.0.1:8000/.
6) Click login and enter a username and password (Project is implemented to login with any username and password)


Run test case for implemented view:
1) Open a command prompt and navigate to the project directory
2) python manage.py test
